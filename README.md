This project is a fictional criminal profile database representing iTab, a mock government agency. It features several profiles of original characters, and case files that were designed to build into a larger narrative.

You can find the currently deployed version at https://james-york2008.github.io/criminalProfileDatabase/

All of the narrative content was created by OpenAi's ChatGPT, with the direction and fine-tuning done by James York. All of the code and design were done by James York in the most recent version.

For transparencys sake, in earlier versions ChatGPT also made the css, but as of March 22, 2026 all of ChatGPT's css has been replaced.
